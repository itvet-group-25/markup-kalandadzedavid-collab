# საშინაო დავალება — გაკვეთილი 1: HTML-ის საფუძვლები

**სირთულე:** დამწყები  
**ვადა:** შემდეგი გაკვეთილი  
**ფაილები:** შექმენი `homework/` საქაღალდე და მასში მოათავსე ყველა ფაილი

---

## დავალება 1 — HTML-ის ძირითადი სტრუქტურა

შექმენი ფაილი სახელად `index.html`.

ფაილი უნდა შეიცავდეს HTML-ის სრულ, სწორ სტრუქტურას:

- `<!DOCTYPE html>` დეკლარაცია
- `<html>` ელემენტი `lang` ატრიბუტით (მაგ. `lang="ka"`)
- `<head>` სექცია, სადაც მითითებულია:
  - სიმბოლოების კოდირება (`<meta charset="UTF-8">`)
  - გვერდის სათაური (`<title>`) — დაუწერე საკუთარი სახელი
- `<body>` სექცია

> **შენიშვნა:** `<body>`-ის შიგთავსი შეავსე შემდეგი დავალებების შესრულებისას.

---

## დავალება 2 — სათაურები (h1–h6)

`index.html`-ის `<body>`-ში დაამატე 6 სათაური (`<h1>`-დან `<h6>`-მდე).

**თემა:** შენი საყვარელი წიგნი, ფილმი, ან თამაში.

მაგალითად:

- `<h1>` — მთავარი სათაური (სახელი)
- `<h2>` — ჟანრი
- `<h3>` — ავტორი / რეჟისორი
- `<h4>` — გამოშვების წელი
- `<h5>` — მოკლე აღწერა
- `<h6>` — შენი შეფასება

---

## დავალება 3 — პარაგრაფები

სათაურების ქვემოთ დაამატე **მინიმუმ 3 პარაგრაფი** (`<p>`) შენი საყვარელი წიგნის, ფილმის ან თამაშის შესახებ.

- პირველი პარაგრაფი: მოკლე მოთხრობა / სიუჟეტი
- მეორე პარაგრაფი: რატომ მოგწონს
- მესამე პარაგრაფი: ვის ურჩევდი

---

## დავალება 4 — ტექსტის ფორმატირება

მესამე პარაგრაფში (ან ახალ პარაგრაფში) გამოიყენე **ყველა** შემდეგი ტეგი **მინიმუმ ერთხელ**:

| ტეგი       | გამოყენება                     |
| ---------- | ------------------------------ |
| `<strong>` | ძალზე მნიშვნელოვანი სიტყვა     |
| `<b>`      | გამოყოფილი სიტყვა (სტილისტური) |
| `<em>`     | ხაზგასმული, ემოციური სიტყვა    |
| `<i>`      | ტერმინი ან უცხოური სიტყვა      |
| `<mark>`   | გამოკვეთილი / მონიშნული ტექსტი |
| `<small>`  | განმარტება ან წვრილი ტექსტი    |
| `<del>`    | გადახაზული სიტყვა              |
| `<ins>`    | ახლადჩასმული სიტყვა            |

---

## დავალება 5 — ხაზის გადატანა (`<br>`)

შექმენი ახალი ფაილი სახელად `address.html`.

ამ ფაილში დაწერე შენი (ან გამოგონილი) სრული მისამართი — **ერთ პარაგრაფში** — `<br>` ტეგების გამოყენებით თითოეული სტრიქონის გასაყოფად.

**სტრუქტურა:**

```
სახელი გვარი<br>
ქუჩა, № X<br>
სახლი / ბინა X<br>
ქალაქი, საქართველო<br>
```

---

## დავალება 6 — ქვედა და ზედა ინდექსი (`<sub>` და `<sup>`)

შექმენი ახალი ფაილი სახელად `formulas.html`.

ამ ფაილში გამოიყენე `<sub>` და `<sup>` ტეგები და დაწერე:

**`<sub>` გამოყენება (ქვედა ინდექსი):**

- წყლის ფორმულა: H₂O
- ნახშირორჟანგი: CO₂
- გლუკოზა: C₆H₁₂O₆

**`<sup>` გამოყენება (ზედა ინდექსი):**

- კვადრატი: x²
- კუბი: y³
- ეინშტეინის ფორმულა: E=mc²
- სიმძლავრე: 2⁸ = 256

## დავალება 7 (ბონუს) — "ჩემი პროფილი" გვერდი

შექმენი ფაილი სახელად `profile.html` — **საკუთარი პროფილის გვერდი**, სადაც გამოიყენებ ყველა ამ გაკვეთილში ნასწავლ ტეგს.

გვერდი უნდა შეიცავდეს:

- სახელი და გვარი (`<h1>`)
- მოკლე შესავალი წინადადება (`<p>`)
- ინტერესები / ჰობი (`<p>` + ფორმატირების ტეგები)
- საკონტაქტო მისამართი (`<br>`-ით)
- ერთი სამეცნიერო ფაქტი ან ფორმულა (`<sub>` / `<sup>`)
